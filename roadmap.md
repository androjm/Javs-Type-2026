## 📅 Fase 2: JavaScript Moderno - El Motor (Mes 3-5)
> *Entender el "por qué" antes que el "cómo".*

### 1. Conceptos Base Profundos
* **Memoria:** Diferencia entre **Stack** (primitivos) y **Heap** (referencias).
* **Contexto:** Execution Context, Scope Chain y Closures.
* **Mecánica:** El motor V8 y el Event Loop (Microtask vs Macrotask queue).

### 2. Lógica ES2026+
* **Asincronía:** Promesas, `async/await` y `Array.fromAsync()`.
* **Estructuras:** Map, Set, WeakMap, WeakSet.
* **Novedades:** Temporal API (fechas) y `using` (manejo de recursos).

## 📅 Fase 3: TypeScript - El Estándar Industrial (Mes 6-7)
> *En 2026, el tipado no es opcional.*

* **Tipado Esencial:** Interfaces, Type Aliases y Uniones Discriminadas.
* **Tipado Avanzado:**
    * **Generics ($<T>$):** Código reutilizable con seguridad de tipo.
    * **Utility Types:** `Pick`, `Omit`, `Partial`, `Record`.
    * **Mapped & Conditional Types:** Lógica dentro del sistema de tipos.
* **Validación:** Uso de **Zod** para asegurar datos de entrada (Runtime validation).
